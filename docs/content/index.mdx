# Outline

## Introduction

### What is Gutenberg

Gutenberg is the new default WordPress content editor. Gutenberg introduces a block based approach to editing that aims to makes it easier to work with media rich pages and posts.

### What is Gatsby WordPress Gutenberg

Gatsby WordPress Gutenberg is a framework that makes it possible to work with Gutenberg blocks in Gatsby. Gatsby WordPress Gutenberg provides a live Gatsby preview of posts and pages as you create or edit them in Gutenberg.

### Features

- Live preview WordPress Gutenberg posts and pages in Gatsby
- Render Gutenberg blocks with custom React components
